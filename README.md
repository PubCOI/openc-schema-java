## OpenCorporates POJOs

POJOs derived from schemas at https://github.com/openc/openc-schema

### Building

```bash
mvn clean install
```

### Importing

```xml
<dependency>
    <groupId>org.pubcoi.schemas</groupId>
    <artifactId>openc-schema</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
